# ♻️ EcoPoint
EcoPoint is a gamified web-based waste management platform that integrates Artificial Intelligence, Computer Networks, and Cloud Computing into a single scalable system.

The platform encourages users to manage waste responsibly by combining smart waste classification, real-time cloud infrastructure, and reward-based gamification mechanisms.

EcoPoint is designed to:
1. Promote sustainable waste management behavior
2. Utilize AI for automatic waste classification
3. Leverage cloud infrastructure for scalability and storage
4. Apply networking principles for distributed system communication
5. Implement gamification to increase user engagement
This project demonstrates the integration of:
1. Artificial Intelligence
2. Computer Networks
3. Cloud Computing
